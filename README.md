# Warsztaty Clean Code

Repo do ćwiczeń: Clean Code, SOLID, TDD

## Zadanie Clean Code: 


Zmień branch:

```
git switch clean-code-task
```

Kompilacja (z poziomu `wokrshop\`):

```
scripts\compile.bat
```

Uruchomienie (z poziomu `wokrshop\`):

```
java -cp out clean.LongMethodApp
```
